Rest

1 url + 1 "verbe" → Action connue sur une ressource

"Verbes" : GET, POST, PUT, DELETE

  • GET "/articles" → Liste des articles
  • POST "/articles" → Création d'un article
  • GET "/articles/:id" → Lecture de l'article précisé
  • PUT "/articles/:id" → MAJ de l'article précisé
  • DELETE "/articles/:id" → Suppression de l'article précisé
